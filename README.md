# FIFA World Cup 2026 - Group H Analysis

Data analysis project focused on the FIFA World Cup 2026 Group H teams using Python, Pandas, data visualization, and football performance metrics.

This project analyzes the recent performances of:

- Spain
- Uruguay
- Saudi Arabia
- Cape Verde

The analysis is based on the last 10 matches played by each national team, including official matches and international friendlies, with the goal of evaluating team form, offensive efficiency, defensive stability, and projected group-stage performance.

---

# Project Objectives

- Analyze recent team performance before the 2026 FIFA World Cup
- Compare offensive and defensive metrics between Group H teams
- Create visual dashboards and automated reports
- Generate data-driven projections for the group stage
